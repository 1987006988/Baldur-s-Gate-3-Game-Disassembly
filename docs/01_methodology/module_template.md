# Module Template

以下模板适用于 `docs/03_analysis/` 中的大多数模块。

## Question

本模块要回答什么具体问题？

## Short answer

用 3 到 6 句话给出当前最短答案，并标明哪些部分是事实、哪些是推断。

## Player-layer observations

- 玩家看到了什么
- 玩家在何处感到自由、沉浸、被反馈
- 哪些案例最能代表这一体验

## System-layer explanation

- 哪些规则、资源、状态或流程在起作用
- 这些系统如何耦合
- 它们如何产生前述体验
