# Core Thesis

## Question

BG3 的高反应性体验，究竟主要来自什么样的系统缝合？

## Short answer

当前判断是：BG3 的核心优势不只是“分支很多”，而是把任务状态、同伴关系、长休节奏、战斗与环境解法、区域推进结构放进同一个可回流系统里，让玩家频繁感到“我刚才的选择被世界记住了”。这是一个以系统回流为核心、以电影化包装放大感知强度的设计组合。上述表述目前主要属于推断，仍需用更多官方与案例来源压实。 

## Current thesis

- `推断`：玩家的自由感来自“可尝试解法多”与“结果会被后续内容回应”同时成立，而不只是即时分支数量。
- `推断`：BG3 把 D&D 式规则空间转化为数字 CRPG 的关键，不仅是规则移植，而是把规则选择、叙事触发和空间入口整合成持续反馈链。
- `待验证问题`：官方在多大程度上明确把“反应性”作为跨模块统一设计目标，而非宣传层关键词。
