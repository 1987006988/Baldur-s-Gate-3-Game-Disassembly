# Macro Structure

## Question

BG3 如何通过 Act、区域、入口与推进结构，让玩家感到自己在主动穿行一个开放而非线性的世界？

## Short answer

当前判断是，BG3 的宏观结构不是纯开放世界，也不是严格线性章节，而是由多个高密度区域、多个入口、交叠目标和阶段性收束组成。玩家感受到的自由，一部分来自同一目标可由不同路径接近，另一部分来自区域内任务与战斗空间的重叠布局。具体结构仍需更多官方资料与案例支持。

## Player-layer observations

- 玩家通常不会只沿一条主线推进，而是在区域探索、遭遇、支线和主线目标之间来回切换。
- 多入口、多垂直空间、多可绕行路径让“我自己找到办法”的感觉更强。

## System-layer explanation

- 区域推进与任务状态并行存在，允许玩家先接触后补做。
- 入口设计与空间结构让同一目标支持侦察、对话、潜行、战斗等多种靠近方式。

## Implementation-layer evidence / hypothesis

- 已有证据：待补官方对区域设计、自由推进、场景响应的表述。
- 合理推断：高密度区域优于大而空的开放地图，更适合承载任务回流和多入口设计。
- 待验证问题：Act 切换在多大程度上是系统收束点，而不仅是剧情章节划分。
