# Source Note

## Source ID

`BG3-OFF-005`

## Title

Community Update #28 Closed Beta

## Type

official community update

## Why this source matters

这是目前与 `Dark Urge 营地 bard 事件` 关联最直接的官方来源。它首次把“某位 bard 会在营地以可控制角色身份暂时加入”公开说出来，说明这条事件至少不是纯玩家误解或纯社区传闻，而是被官方作为 Patch 7 / Dark Urge 改动点明示过。

## Reliable facts

- 官方在 Community Update #28 中写道：如果你在扮演 Dark Urge，`a certain bard would now be able to join the party as a controllable character at camp if there’s an empty slot, TEMPORARILY.`
- 后续官方更新再次回指 Community Update #28，并补充强调：`There are no new companions. There is only death.`
- Community Update #28 与后续 Patch 7 文案都把该变化放在 Dark Urge 相关改动语境中，而不是普通同伴招募语境中。

## Useful inferences

- `推断`：这条 bard 事件至少包含“营地临时入队”这一可被官方直接确认的状态变化，而不只是社区主观总结。
- `推断`：官方用 “TEMPORARILY” 和 “There is only death” 这样的措辞，暗示该事件的后续反馈并非稳定 companion 路线，而是一次带强后果的短暂状态转移。

## Limits and bias

- 这条来源没有直接写出 bard 的姓名、触发条件、后续后果细节或所有分支。
- 它更像 spoiler-aware tease，而不是完整 patch note，因此不能单独重建整条事件流程。
- 官方的戏谑措辞需要谨慎解读，不能把语气本身当作事实。
