# Source Note

## Source ID

`BG3-OFF-006`

## Title

Community Update 14: Forging the Arcane

## Type

official community update

## Why this source matters

这是目前与 `Grymforge / Grym 环境解法路径` 最接近的高层官方来源。它虽然不直接点名某个具体 cheese 或机关解法，但明确把 Grymforge 定义为包含 `new questlines, choices, ... complex combat encounters` 的区域，同时把新武器动作、空间与战斗战术扩展一起推出。对当前仓库来说，它足以把 Grymforge 从“只有社区案例在谈”提升到“有官方高层框架支撑”的层级。

## Reliable facts

- 官方在 Community Update 14 中把 Grymforge 描述为一个包含 `new questlines, choices, cinematics, complex combat encounters` 的新区域。
- 同一更新明确推出了新的 Weapon Actions，强调要让近战与远程战斗拥有更多 `tactical agency`。
- 更新还说明高地规则被调整、战斗 UI 与目标预览被优化，表明团队在同一阶段同时推进空间判断、环境可读性与战斗决策表达。
- 更新中的 patch notes 还写到 `Added new Grymforge region ... Added setup for several Grymforge quests in the Underdark.`，说明 Grymforge 从一开始就被放在“任务 + 战斗”共同推进的语境下。

## Useful inferences

- `推断`：如果要在任务模块中加入一个“战斗 / 环境绕解”补强案例，Grymforge 是比一般随机关卡更合适的对象，因为官方一开始就把它定义成任务与复杂战斗共同构成的区域。
- `推断`：Grymforge / Grym 的更稳落点应先在 `04_combat_and_environment` 深挖，再反向回流到 `02_quests_and_choices`，因为当前官方来源仍主要支撑“复杂战斗框架”，而不是某条具体任务后果链。

## Limits and bias

- 这条来源只提供高层区域与战斗设计框架，不能直接证明某个具体环境解法一定是官方重点支持路径。
- 它来自较早阶段的社区更新，和正式版具体执行之间可能存在差异。
- 它适合支撑“为什么选 Grymforge 作为案例方向”，不适合单独承担具体路径结论。
