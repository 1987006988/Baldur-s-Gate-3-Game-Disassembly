# BG3 Official Sources

本页用于整理官方来源类型，而不是简单堆链接。

## 优先收集的官方来源

1. 官方网站与产品概述
2. Larian 社区更新、补丁说明、热修说明
3. 开发者访谈、Panel from Hell、FAQ
4. 官方视频、Podcast、演讲或设计表述
5. 官方 Modding / Journal / Osiris 文档

## 每类来源要提炼什么

### 官方产品页

- 官方如何定义 BG3 的核心卖点
- 官方如何表达区域跨度、选择后果、队伍协作与电影化叙事

### 补丁 / 热修 / Community Update

- 哪些修复条目暴露了任务状态、营地 scene、companion reaction、flow、dialogue accessibility 等系统边界
- 哪些条目最适合当 Implementation Validation 的公开入口

### 开发者表述

- 团队如何谈多入口、非常规 problem-solving、路径覆盖和后续承接

### 官方 Modding / Journal / Osiris 文档

- Quest / Objective / Step 的公开结构是什么
- Journal 如何组织和显示任务层级
- Osiris 如何描述事件、查询、调用、数据库与规则
- 哪些文档可以帮助我们把“体验判断”升级为“公开可验证的结构判断”

## 整理时注意

- 先登记 Source ID，再做摘录
- 同一来源如果很长，优先提炼可支撑模块结论的部分
- 官方宣传语只能说明“设计意图”，不能自动等于“玩家实际体验”
- Modding 文档是验证入口，不是 shipped content 私有脚本的自动实锤
