# BG3 Official Sources

本页用于整理官方来源类型，而不是简单堆链接。

## 优先收集的官方来源

- 官方网站与产品概述
- Larian 社区更新、补丁说明、热修说明
- 开发者访谈、GDC 或 postmortem 类分享
- 官方视频、Panel from Hell、FAQ

## 每类来源要提炼什么

- 官方如何定义游戏核心卖点
- 官方如何描述选择、反应性、同伴、战斗、环境互动
- 补丁中哪些措辞能暴露系统结构、状态管理或优先级
- 哪些术语能成为实现验证层的锚点

## 整理时注意

- 先登记 Source ID，再做摘录
- 同一来源如果很长，优先提炼可支撑模块结论的部分
- 官方宣传语只能说明“设计意图”，不能自动等于“玩家实际体验”
