# Source Note

## Source ID

## Title

Patch #2 Now Live!

## Type

official update / patch notes

## Why this source matters

这条来源直接把“营地不是纯休息容器”推到更稳的位置。它一方面新增了 `Withers' Wardrobe of Wayward Friends`，说明营地可以承载队伍状态与角色出入的系统处理；另一方面又给 Karlach 增加了跨 Act 的反思时刻，说明营地 / 休息节点也承担叙事反馈与关系推进的落点。

## Reliable facts

- Patch 2 正式引入了 `Withers' Wardrobe of Wayward Friends`，允许玩家在营地处理合作模式下的队伍成员去留。
- 同一补丁新增了 Karlach 结局相关场景，并补入了她在 Acts 1 和 2 之间反思引擎状态的新时刻。
- 补丁还新增了一个 Avatar Karlach 在第二次升级后反思引擎可能烧毁的时刻。

## Useful inferences

- `推断`：营地至少不是单纯的资源重置界面，而是角色状态、队伍构成和叙事反思被集中展示的系统节点。
- `推断`：当 Larian 选择把 Karlach 的“反思时刻”补在跨 Act 的间隙里时，说明营地 / 休息节点本身就是设计中的反馈折返点，而不是后期附加对白容器。

## Limits and bias

- 这条来源对 camp / rest 的支持主要是高层结构支持，不直接给出完整触发链。
- `Withers' Wardrobe` 更偏系统功能，不能单独证明所有营地反馈都按同一逻辑组织。
- Karlach 的新增时刻具有角色特定性，不能直接外推成所有同伴的普遍规律。
