# ExecPlan: Act 2 Total Framework

## Goal

把 `Act 2` 固化成一块“中盘压力 / 反应性重组框架包”：先用已有官方来源与 Journal 公开结构语言，说明它不是影咒地区氛围摘要，而是 BG3 把前史状态、营地反馈、任务推进与阶段收束更紧地绑在一起的中盘桥梁；再把稳定结论最小回写到宏观结构、营地 / 同伴与实现验证模块。

## Why it matters

- 如果这一步写成“影咒森林很压抑”的气氛摘要，仓库会在刚进入 `Act 2` 时重新退回剧情复述。
- `Mountain Pass / Creche` 已经证明 late `Act 1` 会用门槛与伙伴张力给中盘加压；`Act 2 总框架` 要继续回答的是，这种压力如何变成阶段级结构，而不是局部情绪。
- 先把上位框架锁稳，后续 `Shadow-Cursed Lands`、`Last Light / Moonrise / Gauntlet` 才不会各自散开成地点导览。
