# ExecPlan: Shadow-Cursed Lands Region Pack

## Goal

把 `Shadow-Cursed Lands` 固化成 `Act 2` 的第一块具体区域压力组织器：先用已有 `Act 2` 上位框架与官方高层来源锁定“这块证明的不是影咒气氛，而是风险如何重组探索、推进与反馈读取”，再新建聚焦压力循环的 case note，最后把稳定结论最小回写到宏观结构、战斗 / 环境、任务与选择、实现验证模块。

## Why it matters

- 如果这一步写成“影咒地区很压抑”的氛围摘要，`Act 2` 刚建立的“中盘压力 / 前史重读框架”就会立刻退化成区域导览。
- `Act 2 总框架` 已证明中盘不是“更压迫 = 更线性”；`Shadow-Cursed Lands` 现在必须把这层上位判断落成第一个可复述的区域组织器。
- 只有先做稳这块，后续 `Last Light / Moonrise / Gauntlet` 才能被写成收束点组，而不是各自平行的地点笔记。
